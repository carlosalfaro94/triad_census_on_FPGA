Detailed Description

This file contains the definitions of the functions needed to create and manage a graph.
In our implementation, the graph is read from a file which, in each of its lines, contains an edge of the graph in this format:
src_node_id tgt_node_id
In the structure, nodes can be ordered by ascending order of their ids.

uint32_t  node_id1,
uint32_t  node_id2,
DIRECTION  dir 
)

Function that adds an edge to a certain node. To add an edge e,
we need to have the ids of the two nodes that the e connects; ant its
direction. The function basically adds the edge to the adjacency list of
the node with id node_id1, setting the neighbor id of the edge to node_id2.

Parameters
GRAPH*g: The graph in which we want to add the edge
uint32_tnode_id1: The id of one of the nodes of the edge
uint32_tnode_id2: The id of the other id
DIRECTIONdir: The direction of the edge: if it goes from id1 to id2.
OUT_IN if it goes from id2 to id1.
Returns
STATUS: OK if the insertion was successful. ERR otherwise
