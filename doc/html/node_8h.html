Detailed Description

This file contains the definitions of the functions
needed to create and manage a node of a graph.
In our implementation, the graph is basically a list of nodes
which are identified uniquely with and unsigned integer. Each
node keeps track of its adjacency list, bi means of an array of
edges. Since each edge connects two nodes, edge information will
be duplicated.

union.
